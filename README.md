# Sobre o projeto

Este projeto é uma implementação do tradicional jogo da velha.

# Tecnologias utilizadas
- Java

# Como executar o projeto
Pré-requisitos: Java 11

```bash
# clonar repositório
git clone https://www.github.com/fabriciio95/tictactoe

# abra o prompt de comando na pasta clonada tictactoe e então execute:
java -jar play.jar

```
